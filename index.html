Prefetch

A jQuery plugin to make it easier to prefetch page assets for priming the browser cache. Speeds up the page load times of subsequent page requests.

Download this project as a .zip file Download this project as a tar.gz file

Prefetch Demo

  1. Open your browser's developer tools
  2. Clear your browser cache
  3. Select the network tab to verify prefetched components
  4. Refresh this page

Prefetch times are configurable, and can be individually set for each prefetched item.

Duplicates are handled automatically (good for common include files that are used frequently).
